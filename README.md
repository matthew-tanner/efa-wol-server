# efa-wol-server
> WOL is a workout log api as part of the EFA course work.


## Packages used

- bcrypt
- cors
- debug
- dotenv
- express
- jsonwebtoken
- pg
- pg-hstore
- sequelize

## Usage

- No debugging : ```npm run start```
- macOS Debug[nodemon]   : ```npm run macDebug```
- Windows Debug[nodemon] : ```npm run winDebug```
